Moji is a terminal font finder for people and scripts. It searches enabled
providers concurrently, normalizes font filenames, ranks likely matches, and
downloads through a validation pipeline.

## Install

Choose the package manager you already use:

```sh
npm install -g @microck/moji
pnpm add -g @microck/moji
bun add -g @microck/moji
```

All three commands install the same npm-registry package. It includes binaries
for Linux, macOS, and Windows on x64 and arm64.

## Two ways to work

Run a search in a terminal to open the live interface:

```sh
moji
```

Type a family and press `enter`, or jump directly to results:

```sh
moji "Inter"
```

Use `get` for a non-interactive search, selection, and download:

```sh
moji get "Inter bold"
```

Use `--dry-run` before downloading and `--json` when another program will read
the result.

> [!IMPORTANT]
> A public URL is not proof that a font is licensed for your use. Treat
> `license: unknown` as a prompt to verify the license with the source.
